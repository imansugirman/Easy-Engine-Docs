## Easy Engine Mailhog
Manages mailhog on a site.

### ee mailhog enable

```
# Enable mailhog for site
$ ee mailhog enable example.com
```

### ee mailhog status

```
# Check mailhog status on site
$ ee mailhog status example.com
```

### ee mailhog disable

```
# Disable mailhog for site
$ ee mailhog disable example.com
```
